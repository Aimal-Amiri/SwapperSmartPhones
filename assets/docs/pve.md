## Wat voor webshop ga je maken?
Ik ga een webshop maken waar gebruikers verschillende Apple-producten kunnen kopen, zoals iPhones, iPads, smartwatches en AirPods.

## Welke functionaliteiten biedt jouw webshop?
De functionaliteit van mijn webshop is opgedeeld in twee delen: functionaliteit voor eindgebruikers en functionaliteit die alleen voor de beheerder (admin) toegankelijk is.

Eindgebruikers kunnen producten aan hun winkelwagen toevoegen, de producten in hun winkelwagen bestellen en een melding ontvangen wanneer de bestelling succesvol is geplaatst.

Aan de andere kant heeft de webshop een volledige CRUD-functionaliteit voor de beheerder. De beheerder kan producten toevoegen, bewerken, verwijderen en bekijken. Daarnaast bevat de webshop een resetfunctie waarmee de beheerder alle producten kan herstellen naar hun oorspronkelijke staat. Ook kan de beheerder een overzicht van alle bestellingen bekijken.

Deze functionaliteiten zijn alleen toegankelijk via een beheerdersauthenticatie. Als vereiste binnen het project gebruik ik localStorage om gegevens op te slaan. Hiervoor maak ik gebruik van een JSON-bestand dat enkele gegevens bevat, waarna deze gegevens in localStorage worden opgeslagen.

## Hoe gaat je webshop eruitzien?
Ik probeer de webshop zo veel mogelijk op het prototype te laten lijken. Het prototype is de eerste inspiratiebron en is in Figma gemaakt om een beeld te krijgen van hoe de UI van de webshop ontwikkeld moet worden.

In het docs map is een afbeelding van het prototype geplaatst, zodat het ontwerp bekeken kan worden. Echter, tijdens de ontwikkelingsfase kunnen er extra wijzigingen worden doorgevoerd, waardoor het eindproduct enigszins kan afwijken van het prototype. Het is mogelijk dat ik tijdens het ontwikkelingsproces extra inspiratie opdoe van andere websites.
